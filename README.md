# 🎵SoundFactory 
<br>
SoundFactory는 음악 추천 웹사이트 입니다.<br>
<br>

## 🛠️개발환경
프론트: Create React App기반으로 React를 사용.<br>
백엔드: Node+Express, 통합개발환경: 인텔리제이를 사용.<br>

## 💻주요기능
  음악 추천: 사용자 취향 기반 추천 시스템<br>
  음악 재생: 웹 플레이어를 통한 실시간 음악 재생<br>
  좋아하는 음악 표시: 즐겨찾기 기능으로 선호 음악 관리<br>
  음악 검색: 키워드 기반 음악 검색 기능<br>
  인기 차트: 실시간 인기 음악 순위 제공<br>
  회원가입: 개인 계정 생성 및 맞춤형 음악 경험 제공<br>
